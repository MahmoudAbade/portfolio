## 🚀 The Portfolio

This repository contains the source code for my personal portfolio website. It serves as a central hub where I showcase my projects across Machine Learning, Full Stack Development, and DevOps. The site features a clean, responsive layout built with HTML, Tailwind CSS, and Vanilla JavaScript. It includes:
- A dynamic **Projects Section** displaying my recent work in AI, ML, and Full Stack Development.
- An **About Me** and **Services** section detailing my skills (Python, PyTorch, TensorFlow, AWS, Docker, Kubernetes, React, Node.js).
- A **Hobbies & Interests** segment reflecting my personality beyond code.

## 🛠️ Tech Stack
This web application is built entirely as a static site for blazing fast performance:
- **Frontend Core**: Vanilla HTML5, CSS3, JavaScript
- **Styling Framework**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Google Material Symbols](https://fonts.google.com/icons)
- **Fonts**: [Google Fonts (Poppins)](https://fonts.google.com/)

## 🏃‍♂️ How to Run Locally

If you'd like to clone or view this repository locally, you don't even need a package manager or build tool.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/MahmoudAbade/portfolio.git
   cd portfolio
   ```

2. **Open the project:**
   Simply open `index.html` in your favorite web browser!
   
   *(Optional)* If you prefer, you can serve it via a simple Python HTTP server:
   ```bash
   python -m http.server
   ```
   Then navigate to `http://localhost:8000` in your browser.
